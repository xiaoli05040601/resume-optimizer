# Resume Optimizer | 简历优化助手

不是简历生成器，而是你的求职参谋。

## 它和普通简历工具有什么不同

### 1. 先分析，再动笔

普通工具往往直接开始“写简历”。`resume-optimizer` 会先分析目标 JD 在筛什么人，再把候选人的经历逐条映射到要求上，找出强项、短板和被低估的亮点。

### 2. 它会模拟面试官审你

生成初稿之后，它会切换到招聘方或评委视角，判断：

- 30 秒扫描会不会继续看
- 哪些点最容易直接被 pass
- 哪些经历最值得展开

### 3. 不只适合求职

活动报名、黑客松申请、比赛材料准备也能复用同一套分析框架。把报名规则当成 JD，它就能输出更贴近评委标准的材料。

## 一次调用能得到什么

- 定制简历（中文或英文）
- 匹配度分析
- 模拟招聘方评价
- Cover Letter / 报名文案
- 面试准备清单
- PDF / Word / HTML 导出建议

## 适合的输入

输入可以很粗糙，不需要一开始就很完整：

- 一份岗位 JD 或活动说明
- 任意形式的个人材料
- 旧简历
- LinkedIn / 脉脉资料
- 自述文字
- 项目截图或补充说明

## 最终目标

不是“生成一份看起来像样的简历”，而是让用户在这次投递里，呈现出自己最有胜算的版本。
