# Example Prompts

## 1. Resume Tailoring For A Specific JD

```text
帮我根据这份 JD 优化简历。

【JD】
Senior Product Manager, Merchant Platform
...

【我的背景】
- 3 年电商产品经验
- 做过商家后台工具
- 会 SQL 和基础 Python
...
```

## 2. Hackathon Application Materials

```text
我想报名这个 AI Hackathon，请根据活动要求帮我整理报名材料。

【活动要求】
- 需要项目 idea
- 需要个人简介
- 需要技术背景说明
...

【我的背景】
- 机器学习工程师
- 做过医疗影像 AI
- 有 Gemini API 使用经验
...
```

## 4. Ask For Gap Analysis First

```text
先别急着写简历，先告诉我这份 JD 和我现在的背景差距在哪里，再决定怎么改。
```
