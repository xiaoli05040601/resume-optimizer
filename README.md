# Super Resume

`super-resume` 是一个面向 AI 助手的求职材料定制 skill。它不是单纯“生成一份简历”，而是先分析目标 JD / 活动要求，再结合候选人的原始材料，输出更贴合场景的简历、报名文案和评审建议。

## 适用场景

- 针对具体岗位 JD 定制简历
- 根据活动 / 黑客松 / 比赛要求准备报名材料
- 生成匹配分析、模拟招聘方评价、面试准备清单
- 输出 Markdown 版简历，并支持后续导出 PDF / Word / HTML

## 安装

把整个目录放到你的 skill 目录中即可，例如：

```bash
cp -R super-resume-skill ~/.config/opencode/skills/super-resume
```

如果你的宿主环境使用别的 skills 目录，请按该环境的规范调整路径。

## 触发方式

当用户出现这些意图时建议启用：

- “帮我投这个岗位”
- “根据这个 JD 优化我的简历”
- “帮我写简历”
- “帮我准备黑客松报名材料”
- “我要参加这个比赛，帮我整理报名材料”

## 核心能力

1. 解析 JD / 活动要求，提取 must-have、加分项和关键词
2. 解析用户背景材料，抽取经历、项目、技能和量化成果
3. 输出匹配分析，识别信息缺口并引导补充
4. 生成结构化 Markdown 简历或报名材料
5. 做 self-awareness 分析和模拟招聘方 / 评委评价
6. 给出可直接落地的改写建议

## 设计原则

- 真实性优先，不捏造经历
- 具体优先于泛泛表达
- 让用户对关键改动拥有最终决定权
- 兼顾 ATS 关键词与自然表达

## 兼容性说明

`SKILL.md` 中提到“提问工具”时，指的是宿主环境里的用户追问机制。如果你的环境没有专门的提问工具，也可以直接向用户发出同等语义的问题。
